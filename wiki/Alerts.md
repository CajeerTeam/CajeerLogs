# Оповещения

Оповещения отправляются по правилам, которые фильтруют события по проекту, боту, окружению и уровням.

## Защита вебхуков

Cajeer Logs разрешает только HTTPS-вебхуки и блокирует локальные, приватные и служебные IP-адреса. Для жёсткого режима задайте allowlist:

```env
ALERT_WEBHOOK_ALLOWED_HOSTS=example.com,*.example.com
```

## Антиспам

Поле «Пауза между отправками» ограничивает частоту уведомлений по правилу.


## Очередь доставки

`bin/alert-dispatch.php` только определяет сработавшие правила и ставит webhook-доставки в очередь `jobs`. Фактическая отправка выполняется через `bin/process-jobs.php`. Такой режим не блокирует вычисление правил, если внешний webhook отвечает медленно или временно недоступен.

Рекомендуемые задачи:

```bash
php bin/alert-dispatch.php
php bin/process-jobs.php
```

Ошибочные доставки фиксируются в журнале доставок и могут быть повторно поставлены в очередь через задачи обслуживания.
