# Оповещения

Оповещения отправляются по правилам, которые фильтруют события по проекту, боту, окружению и уровням.

## Защита вебхуков

Cajeer Logs разрешает только HTTPS-вебхуки и блокирует локальные, приватные и служебные IP-адреса. Для жёсткого режима задайте allowlist:

```env
ALERT_WEBHOOK_ALLOWED_HOSTS=example.com,*.example.com
```

## Антиспам

Поле «Пауза между отправками» ограничивает частоту уведомлений по правилу.
